## **한국 보건의료정책 Gray Literature 검색 방법 - Part2. 통합검색**
<div style="text-align:right;">2026.02.20 기준</div>
주의 : 기술적인 부분은 시점에 따라 달라질 수 있으며, 보건의료 중에서 정책 관련이고 한국 자료에만 해당

### **배경**
- 첫째, 한국 보건의료정책 관련 자료 중 Gray Literature는 체계적으로 정리되어 있지 않아 검색이 어려움
	- Peer review된 논문의 경우도 Pubmed에 등재(Pubmed에 등재되는 한국/외국 저널에 실리는 경우)되지 않으면 마찬가지 상황이며 이에 대해서는 따로 정리 예정
	- Gray Literature(회색 문헌) : 학술지나 상업 출판물을 통해 정식 배포되지 않고 비공식 채널로 유통되는 연구·정책 자료. 주로 정부 보고서, 토론회 자료집, 백서, 컨퍼런스 프로시딩, 정책 브리핑 등을 포함
- 둘째, 다양한 LLM과 Search Engine(Google 등)에서 검색이 가능하나, 소위 'LLM이 못 보는 자료'(검색되기 좋은 페이지만 검색하여 결과를 제시하는 편향)가 존재하며 잘 알고 있는 분야를 질의했을 때 검토 Source가 한정된다는 것을 경험할 수 있음
	- 특히 웹페이지 구조가 복잡하고 자료 다운로드와 검색에 로그인(SSO)·권한검증·세션쿠키·토큰만료·캡차·Referer 검사 등이 적용되는 경우 구체적으로 검색방법을 지정하지 않으면 포함하지 않게 됨
- 셋째, 안정적인 아카이빙, 레포지토리가 충분하지 않아 통합적으로 검색할 수 있는 웹페이지가 부족함
	- Part1에서는 기관 중심으로 제시하고, Part2는 통합검색용 DB가 있는 기관을 정리

### **정리 기준 설명**
- 검색 데이터베이스가 있어서 보건의료정책 관련 자료가 있는 웹페이지를 정리
	- 포함 기준은 작성자의 주관적 판단이며 가능한 넓게 정했으며, 보고서를 제공하나 비율이 매우 낮은 경우는 제외
- 포함 자료는 해당 웹페이지에 제시된 내용을 중심으로 작성했으나 대부분이 어떤 자료를 제공하고 있는지 구체적으로 제시하지 않음
- 검색 방법은 기능을 중심으로 기술했으며, 구체적인 검색 방법은 계속 변경될 수 있어 적지 않음
	- 검색연산자 : 조건식 가능, 선택으로 가능, 불가능으로 구분했으며 조건식 가능은 "사과 AND 딸기"처럼 직접 식을 써서 검색할 수 있다는 것이고, 선택으로 가능은 검색 메뉴에서 AND, OR 등을 선택할 수 있다는 것임
	- 검색 필드는 목차까지 검색되는지 여부만 기술했는데, 제목과 키워드 외에 강력한 검색 기능이 될 수 있기 때문임(물론 반대로 필요없는 결과가 많이 나올 수 있음)
	- 상세/고급 검색을 기준으로 기술하였으며, 자료의 종류(학술논문, 보고서, 도서 등)는 웹페이지마다 분류가 상이하여 따로 적지 않음
- 내보내기는 문헌관리도구(Endnote, Refworks, Zotero등)에서 불러오기 쉬운 RIS형식을 지원하는지가 중요하며 엑셀, Text파일로 지원하는 경우도 적음
- Open API : 제공 여부와 상세검색 API 제공 여부로 정리
- 웹페이지가 개편되면 방법은 달라짐(2026.2 기준임)

### **검색 우선 순위**
- 공공 8개, 민간 4개를 정리했고 12개 각각에서 검색하는 것이 가장 확실함
	- 하지만, 중복이 매우 많고 각각의 검색에 시간이 많이 소요되며(검색식 방식이 아닌 메뉴 선택) 검색결과를 내보내기하는 과정에도 시간이 많이 걸림(일부 웹페이지는 내보낼 수 있는 숫자를 제한해서 여러번 해야하는 경우가 있음)
	- 해결방법은 주요 기관에서 제공하는 API를 이용하여 통합 검색하는 웹페이지를 만들고 여기서 Pubmed방식의 검색이 가능하도록 해야하는데, 기술적으로 충분히 가능하나 예산/운영주체 등이 정해져야 하며 누군가(?) 또는 정부에서 진행했으면 함
- AI로 검색하는 것 가능할 수 있으나 현 시점에서 활용이 쉽지 않음
	- 국회도서관 메뉴 선택 검색과 RIS 내보내기 진행
		- Gemini(3 pro) : 검색방법만 안내, 내보내기 불가능
		- ChatGPT(5.2 thingking) : 일부 결과와 검색방법 안내, 내보내기 불가능, Auto Browsing 적용되나 제대로 작동안함(예시로 제공하는 항공권 예약은 느려도 진행하나, 도서관 검색은 잘 진행 못함. 진행한다고 해도 속도가 너무 느림)
		- Claude(pro sonet 4.6) : 일부 결과와 검색방법 안내, 내보내기 불가능, 베타 버전인 Auto Browsing 적용되나 실제 작업은 성공 못하고 느림
		- Perplexity(Sonar) : 일부 결과와 검색방법 안내, 내보내기 불가능. Auto Browsing으로 결과 화면 보여줌(속도는 조금 느리나 정확함)
	- 결과적으로 Perplexity Auto Browsing정도가 자동화라 할 수 있으나 각각 검색하는 것보다 훨씬 비효율적임
	- AI기능의 문제라기보다 국내 웹페이지의 문제일 수 있는 것이, 유사한 작업을 Pubmed에 대해 요청했을 때 원하는 결과를 얻을 수 있었음
	- Pubmed 검색 조건에 맞게 검색하고 검색식과 RIS 제공하기
		- Gemini(3 pro) : 검색식, Pubmed 링크 제시, RIS 제공 못함
		- ChatGPT(5.2 thingking) : 검색식과 Pubmed 링크, PMID포함 리스트, RIS 파일 다운로드 제시함
		- Claude(pro sonet 4.6): 검색식과 Pubmed 링크, PMID포함 리스트, RIS 파일 다운로드 제시함
		- Perplexity(Sonar) : 검색식과 Pubmed 링크, PMID포함 리스트, RIS 파일 다운로드 제시함, Autobrowsing과 대화창으로 요약 등도 제시
	- 제시한 12개 기관 중 AI검색 기능이 있는 곳도 있으나 속도가 느리고 답변이 충분하지 않음
- 우선 순위는 12개 웹페이지 중 어디를 먼저 찾아봐야하는가임
	- 포함하는 자료의 범위를 구체적으로 분석해서 Core를 정하는 작업이 필요하나 학술지와 다르게 기준 설정이 어려워서 주요 정부기관/연구기관을 포함하는 것을 기준으로 설정(전적으로 작성자 검토 결과이고 체계적인 검토를 한 것은 아님)
	- **Core DB(안)** : 국립중앙도서관, 국회도서관, 사이언스온, 프리즘
	- **(검색범위를 넓히기 위한) 부가 DB(안)** : NKIS 국가정책연구포털, 국립의과학지식센터, RISS, 알리오
	- **(기관구독 등으로 접근성이 있으면) 확장 선택 DB(안)**: 민간 기관 4개(DBpia, KISS, e-article, 교보문고 스콜라)
- Core DB
	- 국립중앙도서관, 국회도서관은 납본제가 적용되는 곳으로 가장 우선임
	- 사이언스온은 KISTI(한국과학기술정보연구원)가 운영하며 과학기술 부분에 더 전문성이 있으나 국가R&D연구보고서를 대부분 포함
	- 프리즘은 중앙정부외에도 지방정부 발주 연구가 포함되어 있음
- 부가 DB
	- NKIS 국가정책연구포털은 26개 정부연구기관 보고서를 제공하며 Core에서 검색될 가능성이 높으나 보고서 외 정책자료집, 세미나자료집 등이 있어 유용
	- RISS는 한국교육학술정보원이 운영하며 학술지 위주여서 연구보고서는 상대적으로 적음
	- 국립의과학지식센터는 질병관리청 산하로 의과학 분야에 특화되어 있으며 질병관리청 자료 등이 더 풍부하게 포함되어 있음
	- 알리오는 공기업, 준정부기관, 기타공공기관의 경영공시를 하는 웹페이지로 연구보고서는 공개되는 정보 중 하나로 자체 웹페이지에서 공개하지 않거나 찾기 어려운 자료가 여기에만 공개되는 경우가 있음
- 확장 선택 DB
	- 민간 4개(DBpia, KISS, e-article, 교보문고 스콜라)로 전반적으로 학술지 위주여서 연구보고서 등은 풍부하지 않음

### **기관/단체별 웹페이지 설명**
- 검색연산자 : 조건식 가능, 선택으로 가능, 불가능으로 구분했으며 조건식 가능은 "사과 AND 딸기"처럼 직접 식을 써서 검색할 수 있다는 것이고, 선택으로 가능은 검색 메뉴에서 AND, OR 등을 선택할 수 있다는 것임
	- 검색 필드는 목차나 초록까지 검색되는지 여부만 기술했는데, 제목과 키워드 외에 강력한 검색 기능이 될 수 있기 때문임(물론 반대로 필요없는 결과가 많이 나올 수 있음)
	- 상세/고급 검색을 기준으로 기술하였으며, 자료의 종류(학술논문, 보고서, 도서 등)는 웹페이지마다 분류가 상이하여 따로 적지 않음
- 내보내기는 문헌관리도구(Endnote, Refworks, Zotero등)에서 불러오기 쉬운 RIS형식을 지원하는지가 중요하며 엑셀, Text파일로 지원하는 경우도 적음

**A. Core DB (공공)**

- 국립중앙도서관
	- [https://www.nl.go.kr](https://www.nl.go.kr)
	- 운영 기관 : 문화체육관광부 소속 기관
	- 포함 자료 : 도서, 타도서관 연계자료 등
	- 검색 방법
		- 검색연산자 : 선택으로 가능
		- 검색 필드 : 목차 검색 불가능, 제목, 키워드, 발행자, 저자 검색 가능
		- 기타 특징 : 주제별 검색 가능
	- 내보내기 : RIS가능
	- 오픈 API : 제공(상세검색 포함)

- 국회도서관
	- [https://www.nanet.go.kr/](https://www.nanet.go.kr/)
	- 운영 기관 : 국회
	- 포함 자료 : 도서자료, 국회자료, 멀티미디어, 동영상 등
	- 검색 방법
		- 검색연산자 : 선택으로 가능
		- 검색 필드 : 목차 검색 가능, 제목, 키워드, 발행자, 저자 검색 가능
		- 기타 특징 : 관련기본어 리스트 제공
	- 내보내기 : RIS가능
	- 오픈 API : 제공(상세검색 포함)

- ScienceON
	- [https://scienceon.kisti.re.kr](https://scienceon.kisti.re.kr)
	- 운영 기관 : 과학기술정보통신부 산하 한국과학기술정보연구원
	- 포함 자료 : 국가연구개발사업 수행을 통해 창출된 국가 R&D 보고서와 KISTI 및 연구소, 기업체 등에서 생산한 각종 분석리포트 제공
	- 검색 방법
		- 검색연산자 : 가능(AND/OR는 메뉴 선택이고 다른 연산자는 입력 가능)
		- 검색 필드 : 목차/초록 검색 가능, 제목, 키워드, 발행자, 저자 검색 가능
	- 내보내기 : RIS불가능, Text/엑셀 가능
	- 오픈 API : 제공(상세검색 포함)

- PRISM 정책연구관리시스템
	- [http://www.prism.go.kr/](http://www.prism.go.kr/)
	- 운영 기관 : 행정안전부
	- 포함 자료 : 정책연구보고서(지자체 포함)
	- 검색 방법
		- 검색연산자 : 일부 가능
		- 검색 필드 : 목차 검색 가능, 주제어, 분류, 기관 검색 가능
	- 내보내기 : RIS불가능, 내보내기 불가능
	- 오픈 API : 제공(상세검색 포함, 웹페이지 안내는 없고 공공데이터 포털에 있음)

**B. 부가 DB (공공)**

- NKIS 국가정책연구포털
	- [https://www.nkis.re.kr/](https://www.nkis.re.kr/)
	- 운영 기관 : 국무총리 산하 경제·인문사회연구회
	- 포함 자료 : 경제·인문사회연구회 산하 KDI, 한국교육개발원 등 26개 소관 연구기관 및 3개의 부설기관에서 생산하는 연구성과물, 보고서 외 정책자료집, 세미나 자료 등
	- 검색 방법
		- 검색연산자 : 불가능
		- 검색 필드 : 목차/초록 검색 가능
	- 내보내기 : RIS불가능, 내보내기 불가능
	- 오픈 API : 제공(상세검색 포함)

- 국립의과학지식센터
	- [https://library.nih.go.kr/](https://library.nih.go.kr/)
	- 운영 기관 : 질병관리청 산하 국립보건연구원
	- 포함 자료 : 국가 보건의료기관에서 예산을 투입하여 수행한 연구개발과제의 성과(2010년 이후)
	- 검색 방법
		- 검색연산자 : 일부 가능
		- 검색 필드 : 목차 검색 불가능, 과제명, 키워드 가능
		- 기타 특징 : 상세검색에서는 통합검색이 불가능
	- 내보내기 : RIS불가능, 엑셀 가능
	- 오픈 API : 제공(상세검색 포함)

- 알리오
	- [https://www.alio.go.kr/](https://www.alio.go.kr/)
	- 운영 기관 : 기획재정부
	- 포함 자료 : 기관별 자체연구보고서, 외부용역연구보고서
