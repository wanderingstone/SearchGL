LLM 검색 참고용 사이트 목록(편집/삭제 용이형 텍스트)

[문서 목적]
1) LLM 검색 시 참고하여 답변하도록 업로드하기 위한 파일
2) 기관/단체 단위 블록 구성으로 일부 내용 삭제 용이성
3) RSS, 알리오, 추천 메뉴 관련 정보 제외
4) 아래 포함 사이트 전부 검색 대상 포함(누락 금지)
5) 링크 추측/생성 절대 금지
   - “검색 결과 화면에 실제로 존재하는 URL” 또는 “직접 열어 확인된 URL”만 출력
   - 확인 불가 시 URL 빈 칸 유지 + ‘확인 불가’ 표기

[출력 규칙(LLM용)]
- 검색 범위: 아래 ‘포함 대상’ 목록의 각 사이트(기관/단체) 전부
- URL 출력: 검색 결과/직접 확인 기반 URL만 출력
- URL 미확인: URL 칸 공란 유지, URL 확인 항목에 ‘확인 불가’ 표기
- 본 문서에 없는 하위 링크/문서 링크 임의 생성 금지

[URL 상태값 권장]
- 직접확인: 페이지 직접 열기 기반 확인
- 검색확인: 검색 결과 화면(스니펫/검색결과 URL) 기반 존재 확인
- 확인불가: 검색 결과에서도 근거 부족 또는 접근 차단/오류로 근거 불충분
