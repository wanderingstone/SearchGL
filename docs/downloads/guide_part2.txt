LLM 검색 참고용 통합DB 목록(편집/삭제 용이형 텍스트)

[문서 목적]
1) LLM 검색 시 참고하여 답변하도록 업로드하기 위한 파일
2) DB 단위 블록 구성으로 일부 내용 삭제 용이성
3) 아래 포함 DB 전부 검색 대상 포함(누락 금지)
4) 링크 추측/생성 절대 금지
   - "검색 결과 화면에 실제로 존재하는 URL" 또는 "직접 열어 확인된 URL"만 출력
   - 확인 불가 시 URL 빈 칸 유지 + '확인 불가' 표기

[출력 규칙(LLM용)]
- 검색 범위: 아래 '포함 대상' 목록의 각 DB 전부
- URL 출력: 검색 결과/직접 확인 기반 URL만 출력
- URL 미확인: URL 칸 공란 유지, URL 확인 항목에 '확인 불가' 표기
- 본 문서에 없는 하위 링크/문서 링크 임의 생성 금지

[URL 상태값 권장]
- 직접확인: 페이지 직접 열기 기반 확인
- 검색확인: 검색 결과 화면(스니펫/검색결과 URL) 기반 존재 확인
- 확인불가: 검색 결과에서도 근거 부족 또는 접근 차단/오류로 근거 불충분

[DB 우선순위 안내]
- Core DB: 국립중앙도서관, 국회도서관, ScienceON, PRISM
- 부가 DB: NKIS 국가정책연구포털, 국립의과학지식센터, 알리오, RISS
- 확장 선택 DB(민간): DBpia, KISS, earticle, 교보문고 스콜라

────────────────────────────────────────────────────────
A. Core DB (공공)
────────────────────────────────────────────────────────

[A01] 국립중앙도서관
- 기준 URL: https://www.nl.go.kr
- URL 확인: 직접확인
- 운영 기관: 문화체육관광부 소속 기관
- 포함 자료: 도서, 타도서관 연계자료 등
- 검색연산자: 선택으로 가능
- 검색 필드: 목차 검색 불가능 / 제목·키워드·발행자·저자 가능
- 내보내기: RIS 가능
- 오픈 API: 제공(상세검색 포함)

[A02] 국회도서관
- 기준 URL: https://www.nanet.go.kr/
- URL 확인: 직접확인
- 운영 기관: 국회
- 포함 자료: 도서자료, 국회자료, 멀티미디어, 동영상 등
- 검색연산자: 선택으로 가능
- 검색 필드: 목차 검색 가능 / 제목·키워드·발행자·저자 가능 / 관련기본어 리스트 제공
- 내보내기: RIS 가능
- 오픈 API: 제공(상세검색 포함)

[A03] ScienceON
- 기준 URL: https://scienceon.kisti.re.kr
- URL 확인: 직접확인
- 운영 기관: 과학기술정보통신부 산하 한국과학기술정보연구원(KISTI)
- 포함 자료: 국가 R&D 보고서, KISTI 및 연구소·기업체 분석리포트
- 검색연산자: 가능(AND/OR는 메뉴 선택, 기타 연산자는 직접 입력)
- 검색 필드: 목차/초록 검색 가능 / 제목·키워드·발행자·저자 가능
- 내보내기: RIS 불가능 / Text·엑셀 가능
- 오픈 API: 제공(상세검색 포함)

[A04] PRISM 정책연구관리시스템
- 기준 URL: http://www.prism.go.kr/
- URL 확인: 직접확인
- 운영 기관: 행정안전부
- 포함 자료: 정책연구보고서(중앙정부 및 지자체 포함)
- 검색연산자: 일부 가능
- 검색 필드: 목차 검색 가능 / 주제어·분류·기관 검색 가능
- 내보내기: RIS 불가능 / 내보내기 불가능
- 오픈 API: 제공(상세검색 포함 / 공공데이터 포털을 통해 제공)
